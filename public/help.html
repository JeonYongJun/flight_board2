EastarJet Flight Plotting Board System!

사용법

  • 1 : 그룹웨어 ID와 Password로 접속

  • 2 : 검색을 원하는 날짜와 공항을 지정후 plan 버튼 클릭

  • 3 : 계획 스케줄을 기준으로 정보를 표출하며 운영된 스케줄을 같이 표출하려면 schedule 버튼을 클릭

  • 4 : 게이트 정보를 표출하려면 gage 버튼을 클릭

  • 게이트 정보 : 각 공항의 예정(확정)된 게이트 정보를 배치 작업으로 공사의 홈페이지 정보를 읽어옮.

Request

    1: 정비사 배정 - Departure, Arrival, Board, 각 2명씩 배정 가능, Note 추가 기능

    2: 기재의 Ground Time 에 대한 작업 표시 기능

    3: Daily Check 표시 기능

    4: 추가 정보 표시
